# 8.1 Write Your Name
![Name](austin.png)
## Assignment:
For this assignment, you will create a program that will write your name or any word that is five letters or longer. 

## Rubric
|Description|Points|
|---|---:|
|Creates a Screen object| 1|
|Sets Screen object's background color|1|
|Sets Screen object's title|1|
|Creates at least one Turtle object|1|
|Writes a name or word that is at least 5 letters long|3|
|All letters must be the same size and positioned on one line|3|
|At least one letter is written using coordinates|3|
|At least one letter is written using forward,backward,left,right,setheading|3|
|Line weight should be greater than the default value of 1|1|
|Each letter should be a distinct color|1|
|There is at least one curved line incorporated into the name|2|
|**Total**|**20**|
